Cisco Nexus Telemetry监控的简单实现

发表于 2023-05-24 更新于 2023-05-25 分类于 Telemetry 阅读次数： Waline： 本文字数： 730 阅读时长 ≈ 3 分钟

Telemetry是一种实时监控和收集网络设备状态数据的技术，它可以提供更精细、更准确的网络性能数据，以帮助网络管理员更好地管理和维护网络。为此记录一下简单的实现方式。

InfluxDB安装

RPM包安装方式

  • 下载InfluxDB RPM包
  • yum本地安装

yum在线安装方式

  • 添加软件源
  • yum安装influxdb

influxdb相关设置

  • 安装influxdb客户端压缩包
  • 解压tar包,并拷贝到指定路径
  • 启动influxdb设置开机自启
  • 命令行初始化安装
  • 查看Token

Telegraf安装

  • 添加软件源
  • yum安装telegraf
  • 备份原配置文件
  • 编写新的配置文件
  • 启动telegraf并设置开机自启

Grafana安装

  • 安装grafana-server

  • 设置data source为InfluxDB

    • Name ：自定义名称
    • Query Language ： 选择InfluxQL
    • url ：为influxdb的地址
    • Custom HTTP Headers点击+ Add hearder ：Hearder值填入Authorition,Value值填入下面格式：
    • Database ：influxdb中的数据库名称
    • User ： admin //创建用户名
    • Password ：password //创建的密码
    • HTTP Method : 选择GET
    • 点击Save & test
